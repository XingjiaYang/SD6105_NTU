How to Use

Suggested Order

Start with the map, then check admissions safety, then compare net price and cost, and finally use the profiler and trend views for added context.

Who This Is For

Designed for high school students, parents, counselors, and anyone who wants a clean and intuitive way to compare colleges.

Page Role

This homepage acts as the master navigation layer, while each linked page handles a more focused analysis task.

Visual Style

The layout follows the reference style you showed: light background, rounded panels, generous spacing, dark typography, and soft blue accents in a more compact, polished format.
